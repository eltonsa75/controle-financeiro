# 💰 Controle Financeiro

Um ecossistema full-stack robusto e inteligente voltado para a gestão e análise de finanças pessoais. O sistema combina uma API de alta performance em **.NET**, um front-end dinâmico em **Angular** e persistência de dados em **MySQL**, oferecendo uma experiência fluida para tomada de decisões financeiras através de conceitos de Business Intelligence (BI).

---

## 🚀 Principais Funcionalidades

* 📊 **Dashboard de Evolução Histórica:** Gráficos de linha interativos (Chart.js) controlados por fuso horário local e indexados de forma cronológica contínua por data de emissão.
* 🧾 **Importação Automatizada:** Módulo de leitura e processamento de dados financeiros através de arquivos PDF de notas fiscais ou URLs de competência.
* 🏷️ **Classificação Inteligente:** Sistema interno de categorização automática de produtos e despesas com base em dicionários e mapeamentos dinâmicos.
* 💳 **Cards de KPI Reativos:** Exibição em tempo real de receitas mensais, despesas acumuladas, saldo do período atual e saldo geral consolidado.
* 📅 **Navegação Inteligente:** Carrossel de períodos com travas de escala temporal para consistência na busca de dados históricos.

---

## 🛠️ Tecnologias Utilizadas

### Back-end
* **.NET Core / C#** (Web API)
* **Entity Framework Core** (ORM)
* **LINQ to Objects** (Agrupamentos e modelagem temporal)
* **JWT / Firebase Auth** (Gerenciamento seguro de multi-inquilinos)

### Front-end
* **Angular** (Componentes independentes / Standalone)
* **TypeScript** & **Bootstrap** (UI responsiva)
* **Chart.js** & **chartjs-plugin-datalabels** (Renderização gráfica de dados)

### Banco de Dados
* **MySQL Server** (Persistência relacional)
